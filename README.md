# Real-Time Sales Dashboard (Power BI)

## 📊 Overview
This project demonstrates a simulated real-time sales dashboard using Power BI and Python.

## 🚀 Features
- Sales, Profit, and Quantity analysis
- Category and regional performance insights
- Shipping and delivery analysis
- Simulated real-time data updates

## ⚙️ Tech Stack
- Power BI
- Python
- CSV Dataset

## 🔄 How it Works
A Python script continuously updates the CSV file by adding new data rows.  
Power BI reads this file and refreshes to reflect updated data, simulating real-time streaming.

## 📁 Files Included
- `.pbix` → Power BI Dashboard
- `.csv` → Dataset
- `stream.py` → Python script for data simulation
